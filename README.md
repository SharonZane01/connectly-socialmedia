# Connectly - Developer Social Platform

Connectly is a full-stack social media application designed for developers to share updates, photos, and connect with peers. It features a secure authentication system, real-time post interactions, and cloud-based image management.

## üöÄ Features

* **User Authentication:** Secure Signup & Login using JWT (JSON Web Tokens).
* **Create Posts:** Upload images (via Cloudinary) and add captions.
* **Interactive Feed:** Like and Unlike posts instantly.
* **User Profiles:** Customize profile pictures, edit display names, and manage your own posts.
* **Responsive Design:** Fully optimized for Desktop and Mobile devices.
* **Guest Mode:** Public read-only access for non-authenticated users.
